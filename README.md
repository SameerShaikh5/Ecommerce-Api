# 🛒 E-commerce API

A full-featured backend for an e-commerce application built using **Node.js**, **Express**, **MongoDB**, and various tools like **JWT**, **Cloudinary**, **Redis**, and **Razorpay**. It handles user authentication, product management, cart operations, order management, and payment processing.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 🛍️ Product CRUD operations with image upload (Multer + Cloudinary)
- 🛒 Cart Management (add, remove, quantity control)
- 📦 Order Management (COD & Online)
- 💳 Razorpay Payment Gateway Integration
- 🎟️ Coupon System (create, apply, update)
- 🧠 Redis caching support
- 🛠️ Admin Panel endpoints for user and order management

---

## 🛠️ Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose
- **Authentication**: JWT (JSON Web Token)
- **Image Upload**: Multer & Cloudinary
- **Payments**: Razorpay
- **Caching**: Redis
